# Wireless Auditor Pro 🛡️

A Python-based automation tool for wireless network auditing on Linux.

## 🚀 Features
- **Automatic Interface Detection:** Quickly identifies compatible wireless cards.
- **One-Click Monitor Mode:** Automated switching via `airmon-ng`.
- **Optimized for Hardware:** Designed for the Intel AX200 wireless card on ThinkPad T490s.

## 🛠️ Requirements
- **OS:** Linux (Ubuntu/Kali preferred)
- **Tools:** Aircrack-ng suite (`sudo apt install aircrack-ng`)
- **Permissions:** Root/Sudo privileges

## 💻 Usage
```bash
sudo python3 auditor.py
```
